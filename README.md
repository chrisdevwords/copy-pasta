copy-pasta
==========


Command line tool for copying minified contents of files.

## usage

1. Clone the repo.
2. run npm link
3. in the terminal:
```
copy-pasta -f path/to/my/file.ext
```

### For parameters run:
```
copy-pasta -h
```
